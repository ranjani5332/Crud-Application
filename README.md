# Task Management System
# Project Description

 This is a simple Task Management System developed using Node.js, Express, and MongoDB.
The application allows users to add, view, and delete tasks through a basic web interface.
It demonstrates CRUD operations and backend–frontend integration in a beginner-friendly way.

This project was built as part of a MERN Stack Internship Task to showcase core full-stack concepts.

# Features
- Add a task with title and description
- Display all tasks in a list
- Delete a task
- REST API implementation-
- MongoDB database integration
- Simple and clean user interface
  
# Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- HTML & JavaScript
- Git & GitHub

# CRUD Operations

- Operation  Description
- Create - Add a new task
- Read - View all tasks
- Delete - Remove a task
